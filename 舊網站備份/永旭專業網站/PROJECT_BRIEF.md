# 永旭事務所網站重製專案 — 啟動文件

## 一、專案概要

**事務所名稱**：永旭智慧財產事務所
**英文名稱**：Yong Syu Intellectual Property Office (YSIPO)
**現有網站**：https://ysipo.com.tw/

**專案目標**：重新製作整套網站，主要為了改變定位、強化精品事務所形象、提升信任感、補上新聞動態、智財知識專欄等業界標準模組。

## 二、技術規格

### 開發環境

* 純前端靜態網站（HTML / CSS / JavaScript）
* 不需要 PHP / Node.js 後端
* 響應式設計（RWD）支援桌機、平板、手機
* 預設部署環境：現有的虛擬主機（保留與舊版本相同的部署方式）

### 檔案結構規範

```
/
├── index.html              # 首頁
├── about.html              # 關於我們
├── services.html           # 服務項目
├── insights.html           # 智財知識（部落格首頁）
├── insights/               # 文章存放目錄
│   └── *.html
├── news.html               # 新聞動態
├── contact.html            # 聯絡我們
├── en/                     # 英文版（同樣結構）
│   ├── index.html
│   └── ...
├── assets/
│   ├── css/                # 樣式表
│   │   ├── main.css
│   │   └── ...
│   ├── js/                 # JavaScript
│   │   └── main.js
│   ├── img/                # 圖片
│   │   ├── logo.svg
│   │   ├── office/         # 辦公室照片
│   │   ├── icons/          # 圖示
│   │   └── ...
│   └── fonts/              # 字型（若需要）
└── README.md
```

### 技術棧建議

* **CSS 框架**：可考慮 Tailwind CSS（CDN 載入即可，不需 build process）；或純 CSS 自訂
* **動畫**：使用 CSS Animation + AOS (Animate On Scroll) 函式庫
* **圖示**：使用 SVG icons（Lucide / Heroicons）
* **字型**：繁體中文用 Noto Sans TC / Source Han Sans
* **不使用**：jQuery（已過時）、複雜的 framework

### SEO 要求

* 每頁 `<title>` 與 `<meta description>` 客製化
* Open Graph (og:) 標籤完整
* JSON-LD 結構化資料（LegalService / ProfessionalService schema）
* sitemap.xml 與 robots.txt
* 語意化 HTML（用 `<header>` `<nav>` `<main>` `<article>` `<section>` `<footer>`）

## 九、設計風格指引

### 色彩

* 主色：穩重的深色調（深藍 / 墨綠 / 深灰）
* 強調色：金色 / 銅色（精品事務所質感）
* 背景：純白 / 米白 / 淺灰
* 文字：深灰（#333）/ 黑（#000）

### 字型

* 標題：Noto Serif TC（襯線字，精品感）
* 內文：Noto Sans TC（無襯線，易讀）
* 英文：襯線用 Playfair Display 或 Cormorant Garamond，無襯線用 Inter

### 排版原則

* 大量留白
* 文字行高 1.8 以上（中文閱讀舒適）
* 段落寬度限制 65-75 字（不要全寬撐滿）
* 標題層級清楚（H1 > H2 > H3）

### 視覺元素

* 偏向極簡、有質感
* 不要過多動畫
* 適度使用滾動觸發動畫（AOS）
* 圖示用線條風格 SVG
